# COVID-19 Tracker

![](https://i.imgur.com/dJo9GL1.png)

## Description:

covid19-tracker is a simple app to monitor coronavirus statistics.

### Installation

Use the package manager [npm](https://www.npmjs.com/npm) to install Dependencies:

```bash
npm install
```

### Start Dev Server

```bash
npm start
```

### Production

```bash
npm run build
```
